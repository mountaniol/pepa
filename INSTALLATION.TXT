How to install

1. Clone this project:
# git clone --recursive  git@github.com:mountaniol/pepa.git

2. Switch to the proper branch, today (18/09/2024) it is "pepa-ng":
# git switch pepa-ng

3. Update submodules:
# git submodule update

4. Install dependencies (if you work on a Debian based Linux, like Debian, Ubuntu):
# scripts/install_dependencies.sh

If you run RedHat or other not-DEB based machine, open this script and install listed dependencies using another package manager.

5. Run one the targets:
# make all

or

# make pepa emu

or, to compile a static version of the PEPA:

# make static
